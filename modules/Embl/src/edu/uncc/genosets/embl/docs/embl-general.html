Loading EMBL Annotations

GenoSets provides several ways to load annotations from EMBL files.

Load by selecting from a list of completed EMBL projects

Annotation files can be selected from a list of completed EMBL projects.  GenoSets queries the EMBL database to find all completed projects and the user may select multiple files from a last for loading.  The files are automatically downloaded and stored in the GenoSets database.

Important
Only chromosome files are included in the list of completed EMBL projects.  Plasmid and contig files must use an alternate method to load into GenoSets.

Load by Accession Id

Files may be downloaded from EMBL and loaded into GenoSets by providing an INSD accession number. The accession id may contain a range which is useful for downloading contig sets.  Additionally, multiple accessions may be loaded simultaneously by entering the accession ids separated by line, tab, column, or semicolon.  A single accession id may also include the version number (<accession>.<version>).  The latest version will be loaded if no version number is specified.  If a range is specified, the sequence version will be ignored and the latest version will be loaded.

Example: To download all the contig files for Brucella abortus NCTC 8038, specify the accession range of ACJJ01000001-ACJJ01000045. 

Example: To download plasmid pYJ016 for Vibrio vulnificus YJ016, specify accession AP005352.

Annotation Method

A annotation method is created for each accession id.  The method name is the accession id and the method version is in the format <accession>.<version> of the file.  You do not need to set these in the wizard.
