Loading OrthoMCL Files

Use the Load Data Wizard to load OrthoMCL files.

The user can select the cluster file (usually named groups.txt) and a directory containing all the FASTA files used in the analysis.  The FASTA files are needed to determine singleton clusters as these are not included in the cluster file.  The FASTA files must have the extension .faa or .fasta (case-insensitive).

The cluster file is in the format below:

my_prefix_clusterId: taxon|geneId taxon|geneId
