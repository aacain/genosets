Running OrthoMCL

GenoSets helps to create all the files necessary to run OrthoMCL on a given set of genes.

To create a new OrthoMCL download set:

  • Open the Download window.
  • Create a new Download set of type Annotations.
  • Add all of the annotation files to include in the run of OrthoMCL. You can do this by dragging annotation files from the Project window into the download set.
  • Right-click the File Formats Node and select New File Type.
  • Select OrthoMCL as the new file type to start the OrthoMCL wizard.

Wizard Settings:

  • Path Variables:

    The path variables must be set for the perl bin folder, the orthoMCL bin folder, and the MCL bin folder. The script file can be created local but run on a different machine. Make sure the path variables are for the machine that is to run the script.

    If the location of perl is /usr/bin/perl,  then the perl bin should be set to:

     /usr/bin/

    If the OrthoMCL script orthomclInstallSchema (along with the other orthomcl files) is located at /usr/orthomcl/bin/orthomclInstallSchema, the the orthomcl bin should be set to:

    /usr/orthomcl/bin/

    If MCL is located at /usr/bin/mcl, the the MCL bin path should be set to:

    /usr/bin/

  • BLAST Settings:

    The settings for the BLAST run can be set in the BLAST Settings wizard step. The output format must be set to tab-delimeted (-outfmt 6 for NCBI BLAST) and the name of the BLAST output file should be blast.out.

  • Database Settings:

    OrthoMCL requires an instance of Mysql or Oracle. The database user must have adequate priveleges. See OrthoMCL documentation for more information.

  • OrthoMCL Setting:

    The OrthoMCL settings can be set in the final step of the wizard. For more information on OrthoMCL settings see OrthoMCL documentation.
    Selecting Finish will add the download format to the download set. The files will not be downloaded until the user selects download from the context menu that is available from right-clicking the download set.When downloaded a number of files will be saved on the local machine including a folder named fasta that contains all the fasta files, a config_file, and a script file (runOrthomcl.sh or runOrthomcl.bat).
