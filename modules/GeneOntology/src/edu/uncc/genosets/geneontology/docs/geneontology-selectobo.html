Selecting a Gene Ontology Mapping File

The selected obo file is very important in loading files.  You should select the most recent full obo file.  It is important that all possible terms exist within the obo mapping file.  If the term is not found in the mapping file, an error may occur.  The only time the obo mapping file does not need not need to include all of the terms is when a view is being filtered.

It is recommended that you use FULL/gene_ontology_edit.obolatest.  This file is downloaded from www.geneontology.org/ontology/.
